# Task 1 - Customer & Sales Analysis Dashboard

## 📊 Overview
This project analyzes retail sales data to identify trends, top-performing categories, and customer behavior.

## 📈 Dashboard Features
- Revenue trend analysis (monthly)
- Sales by product category
- Customer segmentation (Gender)
- KPI cards (Revenue, Quantity, Customers)
- Interactive slicers

## 🛠 Tools Used
- Power BI
- CSV Dataset

## 💡 Key Insights
- Sales fluctuate across months with peak periods
- Certain product categories drive most revenue
- Customer demographics influence purchasing patterns

## 🚀 Recommendations
- Focus on high-performing categories
- Improve low-performing segments
- Use seasonal trends for planning
